# Syntax
## Compiler Directives
&nbsp;
### Force Variable Declaration
```vba
Option Explicit
```
&nbsp;
### String Case-Sensitive Comparison
```vba
Option Compare Binary 
```
&nbsp;
### String Case-Insensitive Comparison
```vba
Option Compare Text
```
&nbsp;
### Array Start Index: 0
```vba
Option Base 0
```
&nbsp;
### Array Start Index: 1
```vba
Option Base 1
```
&nbsp;
### Force Module Private
```vba
Option Private Module
```
